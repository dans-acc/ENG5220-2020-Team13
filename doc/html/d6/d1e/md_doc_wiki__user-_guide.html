BeeSafe Project  v1.0
Low-cost tracking device.
User guide

This user guide explains how to use the software.
Note: To exit the project when running, press Ctrl-C

Before use

Before following this user guide, make sure you followed the build guide correctly.

Running the project

Live (Bootloader) run

If building the project with the bootloader for live use, not for development use, please make sure the bootloader build section was followed in the Build Guide. That should result in the project automatically running on the Pi upon booting.

Development run

To run the project as a developer, make sure the complete build guide was followed, not the Bootloader section. After that, the project can be run by navigating into the software/BeeSafePI/build/src folder, and running the main code executable by typing

./BeeSafePI

Alternatively, if just cloned the BeeSafe Github Repository and/or used the installer script, you can run the build script in the root of the repository by typing:

(cd ENG5220-2020-Team13/ && ./runBeeSafe.sh)

Accessing the account services

The website can be found on https://itsbelinda.github.io/ENG5220-2020-Team13/ . While in its current release version does not allow for the account details to be managed through the web interface, the map service visualises the fences and locations on https://itsbelinda.github.io/ENG5220-2020-Team13/map.html .
