Detailed Description

A high-level wrapper for the communication features.

The Comms class acts as a high-level wrapper over the UBlox class, ensuring concurrency-safe access to the hardware. By calling the UBlox commands describing higher level communications functions such as getting a location and sending a text message, and "wrapping it" in a mutex lock it ensures the safe interactions with and operations of the hardware connected to the Raspberry Pi.
