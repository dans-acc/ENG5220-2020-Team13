Detailed Description

The BeeSafe class initiates the complete program.

The BeeSafe class creates the instances of classes required to communicate with the hardware, the monitors overseeing that, and the account information to whom the system must reach out in case of an emergency. Containing the main() function, this is the root of the BeeSafe API.

Attempt to initialise the BeeSafeManager instance responsible for managing the monitor thread, comms and accounts. Note, the function will only set the member pointers in the event all necessary sub-systems are initialised.
