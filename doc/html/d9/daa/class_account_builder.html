Detailed Description

The AccountBuilder class creates and maintains Fence, Account and Contact objects.

The AccountBuilder class is a utility, builder class creating Account, different types of Fence and Contact C++ objects from JSON file data from outside the immediate program. It parses the JSON objects and populates the appropriate C++ objects in the relevant format. It also verifies the correctness of the information relating to each class.
