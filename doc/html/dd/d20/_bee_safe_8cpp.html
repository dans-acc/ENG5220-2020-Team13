void createAccount ( )

A method to create a user account with all necessary details filled in for both the Contacts and Fences objects This is then passed to the program based on which the program verifies the movements of the device, and takes appropriate responses if an alert of incorrect action is received. This method creates a weekly safe time rota, and uses sets of coordinates to create fences for the user. Once the emergency contact is added, the details are used to create the Account object.
